\section{Quickstart}

\begin{lstlisting}[language=bash]
pnpm install
pnpm -r typecheck   # all packages
pnpm test           # vitest workspace (unit / integration / e2e)
pnpm run ssa        # conversational CLI (Ink REPL — see SSA console)

make console        # Palantir-style operator UI on :5173 (+ console-api :4000)
\end{lstlisting}

External dependencies: Redis + Postgres for the backend services. The operator console (\texttt{apps/console/}) runs standalone against fixtures, no DB required.

\section{Full end-to-end SSA pipeline (live infra)}

\begin{lstlisting}[language=bash]
make up             # Postgres (pgvector) + Redis via docker-compose
make migrate        # Drizzle schema + enum prelude + HNSW halfvec(2048) index
make seed           # 33k-obj catalog (CelesTrak SATCAT) + Voyage-4-large
                    # embeddings (~$0.08) + SGP4 narrow-phase screen with
                    # Foster-1992 Pc on regime-conditioned sigma

THALAMUS_MODE=record   make thalamus-cycle   # live LLMs, caches transcripts
THALAMUS_MODE=fixtures make thalamus-cycle   # replay from fixtures/recorded/, zero network
make sweep-run                               # nano-swarm audit + reviewer loop
make hooks-install                           # wire .githooks/pre-commit (spec-check + typecheck gate)
\end{lstlisting}

A fresh run from a clean volume produces \texttt{research\_finding} rows with per-event titles (\texttt{NORAD 28252 $\times$ 38332 --- 2.1 km miss, 2026-04-17T14:12Z, Pc=1.8e-04}), \texttt{research\_edge} rows binding each finding to both satellites, and an \texttt{hnsw halfvec\_cosine\_ops} index serving semantic dedup at 2048-dim.
