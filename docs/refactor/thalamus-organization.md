## TL;DR

- **13 mixed-responsibility offenders** — headlined by nano-swarm (data + duplicate HTTP + utils + class) and executor (dispatch + per-cortex prompts + per-cortex aggregators).
- **8 prompts to hoist** into `prompts/`, one per file.
- **4 type extractions** — biggest win: `explorer/exploration.types.ts` kills the wrong-direction cross-sibling imports.
- **Top ROI extractions**: `callStructuredLlm` (5 sites, 15-30 LOC each), `callOpenAiResponses` (4 sites), `parseXmlEntry` (≥3 RSS/Atom fetchers), `FixtureCache` (2 sites with drift risk).
